# CSharpUI.Avalonia

**CSharpUI.Avalonia** is a fluent, type-safe, XAML-free UI composition library for [Avalonia UI](https://avaloniaui.net/). It enables you to build cross-platform user interfaces entirely in C#, with full IDE support, strong typing, and zero runtime markup parsing.

> 🧱 Build modern Avalonia UIs in **pure C#** — no XAML required.

---

## ✨ Key Features

- ✅ **XAML-Free** UI development
- ✅ **Fluent C# DSL** for defining layouts, controls, styles, and bindings
- ✅ Full **IntelliSense**, **compile-time checking**, and **refactoring support**
- ✅ Ideal for teams that prefer **code-first UI**
- ✅ Dependent on ReactiveUI
